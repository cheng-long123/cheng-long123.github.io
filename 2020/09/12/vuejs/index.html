VUE基础

01-介绍-vue是什么

目标：了解vue的一些基础概念。

官方网站： https://cn.vuejs.org/

官方介绍：渐进式javascript框架。

小概念：

  • 库
    • jquery 是一个js库
    • 只提供一些API给开发者使用
  • UI框架
    • bootstrap layui layer 是UI框架
    • 包含 js html css 主要实现了一套支撑网站开发的常用控件（对话框，弹出框，tab页签…）
  • JS框架
    • vue react angluar 是JS框架
    • 纯js编写，提供了开发网站功能的标准（规则），约定开发规则，通过这些规则组织你的功能代码。

我们理解：vue就是一个js框架。

02-介绍-vue做什么

目标：知道我们学习vue的原因。

vue框架适合做 SPA 类型的网站。

解释两种类型的网站：

  • spa是单页面应用程序（所有网站功能在一个页面上实现），英文：single page application
    • 在访问网站的第一个场景，就加载了所有的资源
      • 缺点：首屏加载过慢
    • 在切换场景的时候不需要再加载资源，切换场景比较流畅。
  • 传统类型网站：切换业务场景是通过页面跳转来实现的，弊端：场景切换速度比较慢，依赖网络。

为什么？

  • 由于单页面应用程序，所有的业务逻辑在一个页面上实现，逻辑复杂度非常高。
  • 我们需要一个框架来组织维护业务功能代码，这个框架就是vue

03-介绍-vue的核心

目标：了解vue的核心功能。

vue提供了那些功能，让我们开发网站更加方便，让我们维护业务功能更加方便。

数据驱动视图

  • 在vue中，通过修改数据，即可更新视图（页面），大大解放了操作dom的时间，把精力放在业务上。

双向数据绑定

  • vue是一个典型的 MVVM 思想的框架
  • M 是数据模型 model
  • V 是视图（界面）view
  • VM 是控制数据模型和视图的管理者
  • 概念：数据可以影响视图变化，视图变化也会影响数据变化。

组件化开发

  • 把一个网站功能，分割若干个组件，相同类型的组件可复用，组件功能是独立的，可维护性高。

04-起步-安装vue

目标：知道怎么去安装vue的几种方式。

不同的阶段使用不同的安装下载方式:

基础阶段：使用script标签引入

  • 本地方式
    • 下载：https://cn.vuejs.org/js/vue.js
    • 引入：<script src="./vue.js"></script>
  • cdn方式
    • 引入： <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>

项目阶段：npm的方式

  • 安装 npm i vue
  • 导入后使用即可

总结：在基础学习阶段，使用本地下载引入的方式。

08-★选项-methods

目标：知道vue选项中methods选项的使用方法和注意事项。

methods选项作用：

  1. 为vue实例提供自定义函数的，大白话：自己的业务函数书写位置
  2. 可以被 vue实例直接调用，且函数的this指向就是vue实例，就是vm
  3. 也可以在vue实例管理的模板（管理的容器|视图）当中使用这些函数
  4. 书写函数有很多方式，在vue中适合哪种方式？ 建议：属性简写

总结：

  • 写函数的地方，给vue实例使用，给模板使用，不建议写箭头函数，属性简写比较合适。

09-术语-插值表达式

目标：了解vue术语插值表达式是什么和使用场景。

在vue中{{ }}语法，叫做：插值表达式，大白话就是输出值得语法。

在插值表达式中可以写：

  • 写data数据字段名称
  • 写methods函数的名称，进行调用
  • 任意js表达式
    • 加减运算
    • 三元运算
    • …
  • 但是不能写js语句
    • 声明变量
    • 分支
    • 循环

10-术语-指令

目标：了解vue术语指令是什么和使用场景。

指令（directive）,其实就是一个以v-开头的标签属性，例如：v-model 双向绑定指令。

vue提供了一些指令，这些指令都是以v-开头，他们可以扩展标签的原有功能。这些指令的值，可以是js表达式，当关联js表达式依赖的数据发生变化，对应的指令就会产生作用。

扩展功能：

  • 输入框双向绑定
  • 标签的显示隐藏
  • 标签显示的内容
  • …

总结：

  • vue提供的v-开头的属性就是指令，扩展标签原有的功能。加上这个指令后，就可以实现某些功能。
